## Tech Stack
- HTML5
- CSS3
- Bootstrap
- Javascript 




## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Our-Rental-Nextjs.git
```

Go to the project directory

```bash
  cd our-rental-nextjs

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
